Mongo.app
=========

Mongo.app is easiest way to run MongoDB on the Mac

### Install

Unzip and copy Mongo.app to /Application directory and double-click.

### Build

For build project you must have installed xCode version >= 5, node.js, npm and grunt.

```
# npm install
# grunt
```
